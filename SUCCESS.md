## 🐛 Troubleshooting

### Server Won't Start?
```powershell
# Clear cache and restart
Remove-Item -Recurse .next
npm run dev
```

### Port 3000 in Use?
```powershell
# Use different port
$env:PORT=3001; npm run dev
```

### Changes Not Showing?
- Hard refresh browser: `Ctrl+Shift+R`
- Clear Next.js cache: delete `.next` folder
- Restart dev server
